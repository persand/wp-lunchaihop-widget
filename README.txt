Widget for www.lunchaihop.se.

== Description ==

This plugin provides a widget for the swedish LinkedIn lunch match making site http://www.lunchaihop.se.

Read more about how to use this widget on http://www.lunchaihop.se/widget.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your side and continue!
